# Propuesta web — Club de Colombicultura l'Eliana

Web estática de una sola página, preparada para abrir localmente o subir a GitHub Pages.

## Antes de presentarla como web oficial

1. Confirmar el nombre jurídico exacto del club.
2. Sustituir el aviso de “propuesta no oficial”.
3. Añadir teléfono, correo, dirección y miembros de la junta.
4. Incorporar calendario y resultados reales.
5. Reemplazar las imágenes provisionales por fotografías propias del club.
6. Conectar el formulario a un correo o WhatsApp oficial.
7. Preparar aviso legal, privacidad y cookies si se recogen datos.

## Publicación rápida

Sube `index.html` a un repositorio público y activa GitHub Pages desde la rama principal y la carpeta raíz.
